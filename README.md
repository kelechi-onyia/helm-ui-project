# Helm UI

This project is opensource for fetching `values.yaml` file, rendering it via react and provides posibility to update the helm values.

## Prerequisitics
The required tools to run this project
- python
- node


> Journal of steps I did
- Install python environment
    : python -m venv ./venv-helm-ui
    : source ./venv-helm-ui/bin/activate
    : pip install -r requirements.txt
- create react app
  : npx create-react-app frontend
  : npm start
